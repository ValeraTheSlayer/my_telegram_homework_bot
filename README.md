# *HomeWork_Bot*
### Описание
Бот, который будет держать меня в курсе о статусе
моей домашки.
### Технологии
Python 3.7

### Запуск проекта в dev-режиме
- Установите и активируйте виртуальное окружение poetry
- В основной директории выполните команду:
```
poetry run homework.py
```
